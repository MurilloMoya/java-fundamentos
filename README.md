# java-fundamentos
Repositório com projetos em Java voltados aos fundamentos da linguagem, lógica de programação, estruturas de controle, vetores, matrizes, funções e modularização.

## Conteúdos abordados
- Estruturas de decisão e repetição
- Vetores e matrizes
- Métodos e modularização
- Entrada e saída de dados
- Resolução de problemas com Java

## Projetos
- Calculadora com menu interativo
- Sistema de cálculo de frete (Transportadora)
- Relatório de vendas
- Relatório de recursos humanos
- Controle de chamados
- Sistema de pontuação (F1)

## Tecnologias
- Java
- IDE: IntelliJ
